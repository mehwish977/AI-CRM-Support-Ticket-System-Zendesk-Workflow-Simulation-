Field Structure Explanation

This section explains the main fields used in the ticket system and how they help in handling customer support queries in a structured way.

Customer Message

This is the original message sent by the customer. It is the most important part of the ticket because everything else is based on understanding this message correctly.

Category

Category is used to group similar types of issues together. For example, refund requests, technical issues, and general inquiries are all different categories. This helps in organizing tickets and makes it easier to track common problems.

Priority

Priority shows how urgent or important a ticket is. Not all tickets need the same level of attention.

High priority is given to issues like login problems or payment-related concerns. Medium priority is used for delays or order issues. Low priority is usually for general questions.

Sentiment

Sentiment reflects the emotional tone of the customer’s message. It can be positive, neutral, or negative.

This helps the agent understand how the customer is feeling. For example, an angry customer may require a more careful and empathetic response.

Intent

Intent is usually suggested by the system using AI. It tries to identify what the customer wants, such as a refund or technical help.

However, it is not always accurate. If the available options do not match the situation, it is better to leave it unassigned rather than selecting the wrong one.

Tags

Tags are short labels added by the agent to describe the issue more specifically. For example, a refund ticket may also include a tag like product_mismatch.

Tags help in filtering and analyzing tickets later.

Assignee

Assignee is the person responsible for handling the ticket. In this project, all tickets are handled by a single support agent.

Final understanding

These fields work together to create a complete view of each ticket. Instead of relying on just the message, the system uses multiple fields to organize, prioritize, and handle customer issues more effectively.
